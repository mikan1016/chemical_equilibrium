化学平衡に関する問題

問題

  1. 化学平衡とはどのような状態か説明せよ。
  2. ルシャトリエの原理に基づいて、次の反応において温度や圧力の変化が平衡にどのような影響を与えるか説明してください：
    N2(g) + 3 H2(g) ⇌ 2 NH3(g) ΔH = -92 kJ/mol
  3. 次の反応の平衡定数(Kc)を求めてください。与えられた平衡状態における各物質のモル濃度は以下の通りです：
    2 NO2(g) ⇌ N2O4(g)
    [NO2] = 0.1 mol/L, [N2O4] = 0.2 mol/L
  4. 次の反応において、Kcの値が0.5であるとき、平衡状態における反応物および生成物のモル濃度の比を求めてください：
    H2(g) + I2(g) ⇌ 2 HI(g)
  5. 以下の反応式に基づいて、反応が右に進むか左に進むか判断してください。初期条件として、[SO2] = 0.5 mol/L, [O2] = 0.5 mol/L, [SO3] = 1 mol/L のとき、Qcを計算し、Kc = 0.2 と比較してください：
    2 SO2(g) + O2(g) ⇌ 2 SO3(g)
  6. 次の反応において、KpとKcの関係式を示し、これらがどのように異なるか説明してください：
    N2O4(g) ⇌ 2 NO2(g)
