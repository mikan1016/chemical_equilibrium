化学平衡に関する解答

解答

  1. **化学平衡**は、可逆反応において、反応が進行しているにもかかわらず、反応物と生成物の濃度が一定に保たれる状態を指します。平衡状態では、正反応と逆反応の速度が等しくなるため、物質の濃度は見かけ上変化しません。
  2. ルシャトリエの原理によれば、反応条件が変化すると平衡がそれに対抗する方向に移動します。
    - **温度が上昇**すると、吸熱反応が優先され、平衡は逆反応（左方向）に移動します。
    - **圧力が増加**すると、分子数が減少する方向に平衡が移動します。この反応では、反応物の総分子数が生成物よりも多いため、圧力を上げると平衡は生成物側（右方向）に移動します。
  3. 平衡定数Kcは、生成物の濃度の積を反応物の濃度の積で割ったものです：
    Kc = [N2O4] / [NO2]2
    Kc = 0.2 mol/L / (0.1 mol/L)2 = 20
  4. H2(g) + I2(g) ⇌ 2 HI(g) の反応において、Kcが0.5である場合、平衡状態では次の式が成立します：
    Kc = [HI]2 / [H2][I2]
    [HI]2 / ([H2][I2]) = 0.5
    この比から、生成物と反応物のモル濃度の比を求めます。
  5. Qc = [SO3]2 / ([SO2]2 × [O2])
    Qc = (1 mol/L)2 / (0.5 mol/L)2 × (0.5 mol/L)
    Qc = 4
    Qc > Kc なので、反応は左に進みます。
  6. KpとKcの関係は、Kp = Kc(RT)Δn で表されます。
    Δnは気体分子数の変化で、この反応の場合、Δn = 2 - 1 = 1 です。
    よって、Kp = KcRT となります。
